# Cosense Graph
This application visualize Cosense page link as Obsidian graph view.


## Setup

```bash
npm install
npm run dev
```
or

```bash
npm run preview:worker
```

## Deployed page

```text
https://graph.kyre.moe/help-jp
```
This page access below endpoint.

```text
/api/pages/:projectName/search/titles
```
